# `@leanup/cli-vanilla`

This package contains the [Vanilla](http://vanilla-js.com/) extension for the [@leanup/cli](https://www.npmjs.com/package/@leanup/cli).

## Installation

`npm install @leanup/cli @leanup/cli-vanilla --save-dev`
